# sdlc-framework

Deterministic, auditable, multi-agent SDLC orchestration framework on top of Claude Code.

Status: **v0.0.0 — substrate bootstrap.** This release ships only `sdlc.__version__`; foundation modules, CLI, and orchestration engine land in subsequent Epic 1 stories.

## Requirements

- Python ≥ 3.10
- [`uv`](https://github.com/astral-sh/uv) ≥ 0.5 for dev environment management

## Quickstart

```bash
uv sync          # resolve and install (uses committed uv.lock)
uv build         # build wheel under dist/
```
