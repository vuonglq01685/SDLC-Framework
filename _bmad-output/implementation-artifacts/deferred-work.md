## Deferred from: code review of 1-13-airuntime-abc-mock-airuntime (2026-05-09)

- **Symlinked `fixtures_dir` or symlinked `*.yaml` files are followed without boundary check** — `src/sdlc/runtime/mock.py:54-65, 113`. `Path.resolve()` follows symlinks; a `fixtures_dir` symlinked outside `tests/fixtures/mock_responses/` will silently load any YAML it points to.
